
# Projeto:

RPA para pesquisa automática de vagas Estágio/Júnior .

## Objetivo:

"Pesquisar vagas de QA Júnior em diversos sites, filtrar por critérios definidos e apresentar uma lista organizada contendo informações importantes e o link para candidatura."

## Stack:

- JavaScript
- Node.js
- Playwright
- Git
- GitHub


## Requisitos Funcionais

* Pesquisar vagas
* Pesquisar em vários sites
* Coletar
* Título
* Empresa
* Localização
* Modalidade
* Salário (quando existir)
* Nível
* Data
* Link
* Eliminar vagas repetidas
* Permitir pesquisa por palavras-chave
